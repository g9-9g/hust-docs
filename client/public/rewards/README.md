# Ảnh quà tặng (FR-11)

Đặt ảnh quà voucher / quà vật lý vào thư mục này. Seed (`server/src/scripts/seed.ts`)
trỏ `imageUrl` tới các đường dẫn `/rewards/<tên-file>` dưới đây.

Khi file chưa tồn tại, UI tự hiển thị placeholder gradient — giao diện không vỡ.

| Quà | Tên file cần thêm |
|-----|-------------------|
| Voucher Highlands Coffee 50k | `highlands-50k.png` |
| Voucher Phúc Long 50k | `phuclong-50k.png` |
| Steam Wallet Code 100k | `steam-100k.png` |
| Thẻ điện thoại 50k | `mobile-card-50k.png` |
| Sổ tay HUST | `hust-notebook.png` |
| Áo thun HUST | `hust-tshirt.png` |

Khuyến nghị: ảnh tỉ lệ ~4:3 hoặc vuông, định dạng PNG/JPG/WebP, kích thước ≤ 500 KB.
